This section lists and introduces the blogs I have translated and summarized during my internship period.

### [Blog 1 - Unified Secrets Security with GitGuardian and AWS Secrets Manager](3.1-Blog1/)
This blog covers the integrated solution between **GitGuardian** and **AWS Secrets Manager** for comprehensive protection of secrets throughout their lifecycle. It analyzes the visibility gap in enterprise security, explains the operating mechanism of the `ggscout` tool, and outlines a 5-step deployment roadmap to help organizations proactively control Non-Human Identities (NHI).

### [Blog 2 - AI-Powered Test Automation: A Breakthrough with Amazon Bedrock and Rapise](3.2-Blog2/)
This blog analyzes the integration of **Amazon Bedrock** and **Rapise** to solve the test script maintenance problem in automation testing. It covers the Self-Healing Scripts mechanism, AI-Driven Test Generation, and a 5-step deployment roadmap that helps QA teams break free from the cycle of manually fixing broken scripts after every UI update.

### [Blog 3 - AWS Transform – When AI Automatically Clears Technical Debt Across Thousands of Repositories](3.3-Blog3/)
This blog introduces the **AWS Transform: Continuous Modernization** feature (currently in preview) — an automated tool that scans, detects, and remediates technical debt across an organization's entire codebase. It covers the 2 operating modes (Continuous & Campaign), out-of-the-box supported features, and integration capabilities with AWS Security Agent and MCP, pointing toward the era of Autonomous Modernization.
